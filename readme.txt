A clean, accessible WordPress block theme with full FSE support, theme.json design tokens, custom block patterns, and WCAG 2.1 AA compliance.

== Description ==

Starter Theme is a modern WordPress block theme designed for accessibility, performance, and ease of use. Built with WordPress Full Site Editing (FSE) in mind, it provides a solid foundation for creating beautiful, accessible websites without compromising on functionality.

**Key Features:**

* **Full Site Editing (FSE)** - Complete theme.json configuration with design tokens
* **Accessibility First** - WCAG 2.1 AA compliant with comprehensive keyboard navigation  
* **Performance Optimized** - Lazy loading, minimal JavaScript, optimized assets
* **Block Patterns** - Hero, testimonials, FAQ, team grid, call-to-action, and more
* **Style Variations** - Ocean, Forest, and Sunset color/typography presets
* **Custom Block Styles** - Enhanced button, image, group, and quote variations
* **Responsive Design** - Mobile-first approach with fluid typography
* **Customizer Integration** - Live preview for colors, typography, and layout options

**Perfect For:**

* Business websites and portfolios
* Blogs and content-focused sites  
* Non-profit organizations
* Educational institutions
* Anyone who values accessibility and performance

The theme includes comprehensive documentation, extensive customization options through the WordPress Customizer, and follows WordPress coding standards for reliability and extensibility.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Search for 'Starter Theme' or upload the theme zip file.
3. Click Install and then Activate.
4. Customize your site through Appearance -> Customize or use Full Site Editing in Appearance -> Editor.

For detailed setup instructions, see the included README.md file.

== Frequently Asked Questions ==

= Is this theme suitable for beginners? =

Yes! The theme is designed to be user-friendly with comprehensive documentation, pre-built patterns, and an intuitive customizer. You can create beautiful websites without coding knowledge.

= Does it work with page builders? =

The theme is built for WordPress's native Full Site Editing (FSE) and block editor, providing the best performance and compatibility. While it may work with page builders, we recommend using the built-in block patterns and customizer for optimal results.

= Can I customize the colors and fonts? =

Absolutely! The theme includes a powerful customizer with color palettes, typography options, and style variations. You can also create custom CSS for advanced styling. All changes are made through the WordPress interface.

= Is the theme accessible? =

Yes, the theme is WCAG 2.1 AA compliant with features like keyboard navigation, screen reader support, proper color contrast, focus management, and reduced motion support.

= How do I get support? =

Check the included documentation first, then visit our GitHub repository for issues and discussions. We provide community support through GitHub.

= Can I create a child theme? =

Yes, child themes are supported. See the README.md file for detailed instructions on creating and using child themes.

== Customization Options ==

**Customizer Sections:**

* **Brand Identity** - Primary, secondary, and accent colors
* **Typography** - Heading font, body font, base font size  
* **Header Options** - Sticky header, search toggle, logo width
* **Footer Options** - Column count, copyright text

* Hero section with cover image and call-to-action buttons
* Testimonials grid with quotes and customer photos

* Call-to-action banners with conversion-focused design
* Featured posts showcase with customizable layouts

== Performance ==

The theme is optimized for speed and Core Web Vitals:

* Lazy loading for images using Intersection Observer
* Minimal JavaScript (< 10KB for core functionality)
* Efficient CSS (< 50KB total) with critical styles inlined  
* Asset optimization with deferred loading and resource hints
* Clean, semantic HTML with minimal DOM manipulation
* Disabled unnecessary WordPress features (emojis, etc.)

== Accessibility ==

Comprehensive accessibility features include:

* WCAG 2.1 AA compliance
* Full keyboard navigation support
* Screen reader optimization with proper ARIA labels
* High contrast mode support
* Reduced motion respect for user preferences
* Color contrast meeting 4.5:1 minimum ratio
* Logical heading structure and landmark roles
* Focus management and visible focus indicators
